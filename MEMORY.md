# 用户偏好和要求

## 报告发送时间
- 每天需要发送两次市场报告：
  - 第一次：中国时间上午9点（Asia/Shanghai 09:00）
  - 第二次：美国东部时间上午9点（对应北京时间22:00）
- 报告仅在工作日（周一至周五）发送
- 报告内容需要包含最新的市场数据，特别是10Y-3M利差等关键指标

## 数据修正要求
- 10Y-3M利差：当数据更新时，需要及时修正报告中的描述
- 最新状态（截至2026-02-24）：利差已转正，倒挂信号显著弱化

## 报告生成标准流程
1. **使用完整10维度模板**：必须包含全部10个分析维度，不得遗漏
2. **数据验证要求**：每次生成报告前必须联网验证所有关键数据的最新值
3. **计算双重验证**：综合评分必须进行双重计算确认准确性
4. **官方数据源优先**：优先使用美国财政部、BLS、美联储等官方数据源
5. **模板标准化**：使用2026-02-25日确认的完整报告模板作为标准
